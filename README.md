# SD2# Circular Fashion Web Application

This project is a full-stack web application focused on sharing, exchange, and mutual aid for fashion items. The theme of the project is "Circular Fashion," promoting sustainability through a community-based approach.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript, PUG (Templating)
- **Backend**: Node.js, Express.js, MySQL
- **Version Control**: Git, GitHub (GitFlow)
- **Containerization**: Docker, Docker Compose
- **CI/CD**: GitHub Actions

## Features
- User authentication (Sign up, login)
- Dashboard to manage shared items (clothes, fashion items)
- Search and browse available items
- Option to exchange or donate items within a community
